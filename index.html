Создавайте
рабочие группы

Пользователи объединяются в рабочие
группы - сообщества.

В сообществах можно:

  • Ставить задачи и поручения сотрудникам
  • Создавать проекты
  • Хранить общие документы
  • Взаимодействовать с другими сообществами

Ставьте задачи
и поручения

Задачи - базовый элемент системы.

Система позволяет:

  • Контролировать исполнение задач
  • Обсуждать задачи по ходу выполнения
  • Отфильтровывать задачи по спискам
  • Просматривать задачи в календаре
  • Создавать чек-листы дочерние задачи
  • Создавать задачи по шаблону
  • Добавлять наблюдателей

Управляйте
проектами

Проект - удобная среда для коллективной работы
над общей глобальной задачей.

В проекте можно:

  • Создавать вехи и задачи
  • Вести обсуждения
  • Прикладывать документы
  • Создавать цепочки задач
  • Подключать внешних специалистов
    и клиентов
  • Сохранять типовые проекты
    в виде шаблонов

Будьте в курсе
Всех событий

Критически важно узнавать
об изменениях в процессах.

Система предоставляет возможность:

  • Настраивать отображаемые
    в системе уведомления
  • Информировать об изменениях с помощью
    смс и e-mail
  • Обмениваться личными сообщениями
